# АИС "Сметный учет и план-фактный анализ строительства"

## Описание проекта
Автоматизированная система на платформе "1С:Предприятие 8.3", предназначенная для учета затрат при строительстве индивидуальных жилых домов. 

### Ключевой функционал:
- **Учет НСИ:** иерархические справочники материалов и объектов.
- **Планирование:** формирование сметной стоимости.
- **Контроль:** регистрация актов выполненных работ и автоматический расчет отклонений.
- **Аналитика:** отчет "План-фактный анализ" с разделением потоков данных через регистры накопления.

### Технологический стек:
- Платформа: 1С:Предприятие 8.3 (версия для обучения).
- Язык: Встроенный язык 1С.
- Модель данных: Объектная метаданная (Справочники, Документы, Регистры).

### Особенности реализации (Интеграция):
В проекте реализован программный модуль для интеграции с MS Excel и PDF. 
*Внимание: В данном репозитории демонстрация результатов экспорта представлена в виде скриншотов из-за ограничений учебной лицензии платформы.*
